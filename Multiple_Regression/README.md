# 📈 Multiple Linear Regression

Welcome to the **Multiple Linear Regression** module! This section explores how to predict a dependent variable based on multiple independent variables.

## 🎯 Overview
While simple linear regression works with a single predictor, real-world problems often involve multiple factors. Multiple Linear Regression (MLR) extends this idea by modeling the linear relationship between several explanatory variables and a continuous response variable.

**Equation:**
`y = β₀ + β₁x₁ + β₂x₂ + ... + βₙxₙ + ε`

Where:
- `y` is the predicted value (dependent variable).
- `x₁, x₂, ..., xₙ` are the distinct independent variables.
- `β₀` is the y-intercept.
- `β₁, ..., βₙ` are the estimated regression coefficients.
